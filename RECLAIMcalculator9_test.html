About This Tool

The RECLAIM Calculator is a specialized parametric design tool for architects and engineers to design and optimize airport baggage handling systems according to IATA standards and regulations.

Automated Calculations

Calculates optimal baggage carousel lengths based on aircraft types and passenger volumes

Compliance Validation

Ensures designs meet international aviation standards (IATA, ICAO)

Space Optimization

Optimizes spatial distribution in baggage reclaim halls for efficiency

BIM Integration

Seamlessly integrates with BIM workflows for documentation

Technical Specifications

  • Built with Grasshopper visual programming environment
  • Compatible with Rhinoceros 3D (v7+)
  • Parametric algorithms for dynamic system adjustments
  • Customizable for various airport sizes and configurations
